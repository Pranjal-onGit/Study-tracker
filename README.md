# Study Tracker

## Overview
The Study Tracker is a comprehensive tool designed to help students manage their study schedules, track their progress, and improve their overall learning efficacy.

## Features
- **Task Management:** Create, edit, and delete study tasks.
- **Progress Tracking:** Visualize your learning progress with graphs and charts.
- **Scheduling:** Set deadlines and reminders for each task.
- **Reporting:** Generate reports of your study habits and progress over time.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Pranjal-onGit/Study-tracker.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Study-tracker
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
To start the application, run:
```bash
npm start
```
